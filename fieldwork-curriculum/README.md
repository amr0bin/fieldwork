# Fieldwork — Accessible Learning Labs

> *Sustainable learning systems are only possible when they move beyond mere legal compliance (accessibility) to embody a radical, intersectional framework (disability justice) that facilitates continuous, context-driven adaptation (emergent learning).*
>
> *This curriculum argues that learning designers working in high-stakes environments are uniquely positioned to close the gap between legislative obligation and genuine, human-centred inclusion — and that closing that gap requires synthesising infrastructure, ethics, and pedagogy simultaneously.*

---

## What this is

This repository is the curriculum backbone for **[Fieldwork](https://www.accessiblelearninglabs.com/fieldwork)**, a 24-week public inquiry published weekly by [Accessible Learning Labs](https://www.accessiblelearninglabs.com).

Fieldwork is designed for inclusive learning designers who are already doing the work and want to think about it more rigorously — alongside others doing the same. Each week produces two posts:

- **Part 1 · Why, See** — the intellectual case: argument, evidence, and a real-world example
- **Part 2 · Try, Share, Reflect** — the practitioner application: an activity, a conversation prompt, and an open question for the field

This repository holds the curriculum architecture behind that public work: the theoretical framework, writing process, templates, concept index, and 24-week map.
